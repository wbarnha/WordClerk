OpenClerk Redaction Checker

Checks a PDF for bad redactions -- a black box drawn over sensitive text without actually removing that text from the document, so it's still present (and copy/paste- or search-able) underneath. Modeled on Free Law Project's x-ray. Everything runs locally in your browser -- the PDF is never uploaded anywhere.

Part of OpenClerk. No file is ever sent over the network -- this page loads a locally vendored copy of pdf.js and does all analysis in-browser.
